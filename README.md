# HR Workforce Analytics Dashboard

## 📌 Project Overview
This project is an interactive HR Workforce Analytics Dashboard developed using Microsoft Power BI. It helps organizations analyze employee data, monitor workforce performance, identify attrition trends, and support data-driven HR decisions.

## 🎯 Objectives
- Analyze employee demographics.
- Monitor employee attrition.
- Track workforce distribution.
- Evaluate job satisfaction.
- Analyze salary and performance trends.
- Support HR decision-making using interactive dashboards.

## 📊 Dashboard Features
- Employee Overview
- Attrition Analysis
- Department-wise Analysis
- Gender Distribution
- Age Group Analysis
- Job Satisfaction Analysis
- Salary Analysis
- Interactive Filters (Department, Gender, Job Role, Education, etc.)

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
